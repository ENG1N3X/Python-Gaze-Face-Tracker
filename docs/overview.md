# Python-Gaze-Face-Tracker — Описание проекта

## Назначение

Python-Gaze-Face-Tracker — система трекинга взгляда и позы головы в реальном времени на Python. Использует веб-камеру для анализа лица, определения направления взгляда, подсчёта морганий и оценки ориентации головы.

Применяется в исследовательских и образовательных задачах: авиация, Human-Computer Interaction (HCI), дополненная реальность (AR), анализ движений глаз.

## Основные возможности

### 1. Детекция лицевых точек
MediaPipe FaceMesh обнаруживает 468 ключевых точек лица на каждом кадре. Минимальный порог уверенности: `0.8` для детекции и трекинга.

### 2. Трекинг радужки глаз
Определяются центры и радиусы радужек левого (`[474–477]`) и правого (`[469–472]`) глаза через `cv2.minEnclosingCircle`. Рассчитывается относительное смещение (Dx, Dy) радужки от угла глаза.

### 3. Детекция морганий
Используется метрика Eye Aspect Ratio (EAR) — отношение вертикальных расстояний между веками к горизонтальному размеру глаза в 3D. Мигание засчитывается, если EAR ≤ `0.51` на протяжении ≥ 2 кадров подряд.

### 4. Оценка позы головы (Head Pose Estimation)
Алгоритм `solvePnP` сопоставляет 6 опорных точек лица (нос, подбородок, углы глаз, углы рта) с 3D-моделью лица. Результат: углы **Pitch** (кивок), **Yaw** (поворот), **Roll** (наклон).

Углы сглаживаются скользящим средним по `MOVING_AVERAGE_WINDOW` кадрам (по умолчанию 10) через класс `AngleBuffer`.

Калибровка (клавиша `c`) фиксирует текущую позу как нулевую точку отсчёта.

### 5. UDP-телеметрия
Каждый кадр отправляется UDP-пакет на `127.0.0.1:7070`.

**Структура пакета (24 байта):**

| Поле | Тип | Размер | Описание |
|---|---|---|---|
| Timestamp | int64 | 8 байт | Метка времени в миллисекундах |
| Left Eye Center X | int32 | 4 байта | X-координата центра левого глаза |
| Left Eye Center Y | int32 | 4 байта | Y-координата центра левого глаза |
| Left Iris Dx | int32 | 4 байта | Смещение радужки по X |
| Left Iris Dy | int32 | 4 байта | Смещение радужки по Y |

Пример: `[1623447890123, 315, 225, 66, -3]`

### 6. Логирование в CSV
При активной записи (клавиша `r`) данные сохраняются в `logs/eye_tracking_log_<дата-время>.csv`.

Колонки: Timestamp, координаты глаз, относительные позиции радужек, счётчик морганий, Pitch/Yaw/Roll (если включено).

## Управление

| Клавиша | Действие |
|---|---|
| `c` | Калибровка позы головы (текущая поза = ноль) |
| `r` | Начать / остановить запись в CSV |
| `q` | Выход из программы |

## Конфигурация

Основные параметры задаются константами в начале `main.py`:

| Параметр | Значение по умолчанию | Описание |
|---|---|---|
| `USER_FACE_WIDTH` | 140 мм | Ширина лица пользователя для масштабирования 3D-модели |
| `DEFAULT_WEBCAM` | 0 | Индекс камеры |
| `SERVER_IP` / `SERVER_PORT` | 127.0.0.1 / 7070 | Адрес UDP-сервера |
| `BLINK_THRESHOLD` | 0.51 | Порог EAR для регистрации моргания |
| `EYE_AR_CONSEC_FRAMES` | 2 | Минимум кадров ниже порога для подтверждения моргания |
| `MOVING_AVERAGE_WINDOW` | 10 | Окно сглаживания углов позы головы |
| `MIN_DETECTION_CONFIDENCE` | 0.8 | Порог уверенности детекции MediaPipe |
| `ENABLE_HEAD_POSE` | True | Включить/выключить оценку позы головы |
| `SHOW_ALL_FEATURES` | True | Отображать все 468 лицевых точек |
| `LOG_DATA` | True | Включить логирование в CSV |
| `IS_RECORDING` | False | Начинать запись автоматически или по клавише `r` |
